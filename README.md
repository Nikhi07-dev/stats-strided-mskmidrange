## ğŸ“Š How It Works

This application calculates the mid-range of a strided array using a mask. The strided array is a sequence of numbers where you can define a step size for selecting elements. The mask lets you filter which numbers to consider in the calculation. 

**Mid-range** is defined as:
\[ \text{Mid-range} = \frac{\text{Max} + \text{Min}}{2} \]

This formula provides a quick summary statistic that gives you the midpoint of the range.
